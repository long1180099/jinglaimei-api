# 腾讯云短信服务接入指南

> 目标：将当前开发模式（验证码打日志）切换为真实短信发送

---

## 一、准备工作（约10分钟）

### 1. 注册腾讯云账号
- 访问 https://cloud.tencent.com 注册/登录
- 实名认证（个人或企业均可）

### 2. 创建短信签名
1. 进入 **控制台 → 云产品 → 短信**
2. 左侧菜单选择 **国内短信 → 签名管理**
3. 点击 **创建签名**：
   - **签名用途**：选择「自用」
   - **签名类型**：推荐选择 **App** 或 **小程序** （需提供小程序AppID）
     - 你的小程序AppID: `wx9ac76bfc2dad7364`
   - **签名名称**：填写 `静莱美` 或 `静莱美平台`
   - **证明材料**：上传小程序截图/备案截图等
4. 提交审核（通常 **30分钟~2小时** 通过）

### 3. 创建短信模板
1. 左侧菜单选择 **国内短信 → 模板管理**
2. 点击 **创建正文模板**：
   - **模板名称**：`登录验证码`
   - **模板类型**：**普通短信**
   - **模板内容**：`{1}为您的登录验证码，有效期5分钟，请勿泄露给他人。`
   - **申请说明**：用于用户手机号登录时发送验证码
3. 提交审核（通常 **30分钟~2小时** 通过）

⚠️ **注意**：
- 模板内容必须和上面完全一致，因为代码中会用到 `{1}` 占位符
- 腾讯云要求模板内容必须符合规范，不能有营销词汇

### 4. 获取API密钥
1. 进入 **访问管理 → API密钥管理**
2. 记录下 **SecretId** 和 **SecretKey**

## 二、配置服务器环境变量

编辑服务器上的 `.env` 文件：

```bash
ssh root@118.195.185.6
cd /root/database-server
vi .env
```

在末尾添加以下3行（填入你实际的值）：

```env
# ====== 腾讯云短信配置 ======
TENCENT_SECRET_ID=你的SecretId
TENCENT_SECRET_KEY=你的SecretKey
TENCENT_SMS_APPID=你的短信应用ID（如1400xxxxxx）
TENCENT_SMS_SIGN=你的签名名称（如：静莱美）
TENCENT_SMS_TEMPLATE_ID=你的模板ID（如1234567）
```

**各参数获取方式**：

| 参数 | 在哪里找 | 示例 |
|------|---------|------|
| `TENCENT_SECRET_ID` | 访问管理→API密钥管理 | `AKIDxxxxxxxxxxxxxxxx` |
| `TENCENT_SECRET_KEY` | 同上 | `xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx` |
| `TENCENT_SMS_APPID` | 短信→国内短信→应用管理（SdkAppId） | `1400123456` |
| `TENCENT_SMS_SIGN` | 你创建的签名名称 | `静莱美` |
| `TENCENT_SMS_TEMPLATE_ID` | 你创建的模板ID（数字） | `1234567` |
