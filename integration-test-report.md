## 五、需要修复的问题

#### PUT /api/users/:id 无法更新 parent_id
- **现象**: API返回 `{ code: 0 }` 成功，但DB中 parent_id 未改变
- **影响**: 整个上下级/差价体系无法工作
- **修复**: 检查 users.js PUT路由SQL；使用独立 POST `/bind-parent` 接口；重新部署最新database-server

### 重新部署 database-server
- 本地代码包含正确的 bind-parent 路由
- 服务器运行的是旧版本代码
