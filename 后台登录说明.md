# 静莱美代理商系统 - 后台登录说明

## 🎯 登录信息概览

你的后台管理系统已成功添加完整的登录认证功能！下面是登录的详细说明。

## 🔑 登录方式

### 1. 直接访问后台系统
- **登录地址**: `http://localhost:3000` (开发环境)
- **默认重定向**: 访问任何后台页面都会自动跳转到登录页面
- **登录后**: 跳转到仪表板页面

### 2. 直接访问登录页面
- **登录页面**: `http://localhost:3000/login`
- **备用地址**: `http://localhost:3000/` (会自动重定向到登录页)

## 📝 登录凭证

### 测试账户信息（模拟数据）
由于当前是开发环境，系统使用模拟数据进行测试：

| 字段 | 测试值 | 说明 |
|------|--------|------|
| **用户名** | 任意3个字符以上 | 如: `admin`、`test`、`张三` |
| **密码** | 任意6个字符以上 | 如: `password`、`123456`、`admin123` |
| **记住我** | 可选 | 勾选后下次自动登录 |

**注意**: 系统使用模拟登录，任何符合格式要求的用户名密码都会登录成功。

## 🔒 认证系统架构

### 1. 前端认证流程
```
用户访问 → 检查认证状态 → 未认证 → 跳转到登录页 → 填写凭证 → 
登录验证 → 保存Token → 重定向到主页面 → 受保护路由访问
```

### 2. 权限控制
- **路由保护**: 所有后台页面都受保护，需要登录才能访问
- **权限检查**: 支持角色和具体权限的检查
- **自动重定向**: 未登录用户自动跳转到登录页

### 3. 用户状态管理
- **本地存储**: 登录信息保存在localStorage
- **自动刷新**: 页面刷新后自动恢复登录状态
- **退出登录**: 清除所有认证信息
